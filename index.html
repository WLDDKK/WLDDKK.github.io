#hexo 博客搭建 node
##node 安装
Node 安装版本很重要！！！推荐安装 nodev14 对应 npm-v6 最不出错
以往的版本 | Node.js (nodejs.org)‘以往的版本’还有两幅面孔无语了，直接点链接和官网依次打开是不一样的。。。。。。
① 进官网选择以往版本
② 选择 node v14 点击 release
③ 选择 v14 x64 msi
④ 一路 next 即可

##安装 cnpm
①“npm install -g cnpm –registry=https://registry.npm.taobao.org”

一直折腾一直是错，卸载安装重复了 3 次，然后莫名其妙好了。。。。
看到 C:\Users\OneAndOnly\AppData\Roaming\npm\node_modules 里竟然有 cnpm！！！脑子觉得应该成功了，再看一遍 cnpm -v 果然出信息了，
然后是用 cnpm 全局安装 hexo：“cnpm install -g hexo-cli”
z 最后 hexo -v 检查安装情况

##Hexo 之旅：
① 新建一个文件夹 blog【文件夹位置 D:\code\blog】无论是 cmd 还是 git 都要从这个位置开始！！！
②“hexo init”有报错，只要文件夹正常就别管
③“hexo s”
打开 http://localhost:4000
④hexo 创建第一篇文章“hexo n MyFirstPassage”
